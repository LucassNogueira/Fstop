# Capstone MVP
## Idea

Create a Formula 1 focused web-app, where users can get information of past races / upcoming races / drivers/ and teams. users will be able to set a favorite driver and team stored in a database. 

## MVP

 - [ ] Users can Register/ Login
 - [ ] Display various data on specific requests
 - [ ] Users can add Drivers to favorite's
 - [ ] Users can add Tracks to favorite's
 - [ ] Users can add Teams to favorite's
 - [ ] Website hoested onto Netlify
 - [ ] Website linked to either PGweb or firebase
